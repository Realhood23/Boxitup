About Enclosure Generator

Create custom 3D printable enclosures for your electronics projects

What is this?

Enclosure Generator is a web application that helps makers, hobbyists, and engineers design custom 3D printable enclosures for their electronics projects. Instead of manually measuring components and modeling boxes in CAD software, you can:

  • Select pre-measured components from our database
  • Toggle which ports and connectors need access holes
  • Visually place components and holes
  • Generate parametric OpenSCAD scripts automatically

How it works

1. Component Database

Our component database is stored on GitHub and contains pre-measured dimensions for popular electronics like ESP32, Raspberry Pi, Arduino, displays, and sensors. Each component includes information about its ports, buttons, LEDs, and other features that might need access holes.

2. Feature Toggles

When you add a component to your project, you can toggle which features need access holes. Don't need the GPIO header exposed? Turn it off. Need USB power but not the SD card? Toggle accordingly. This ensures your enclosure only has the holes you actually need.

3. Visual Editor

Position your components in a visual 2D editor. Place holes on the enclosure faces exactly where you need them. The system tracks which features still need holes and won't let you generate the script until everything is placed.

4. OpenSCAD Output

The generator creates a parametric OpenSCAD script with clear comments and customizable variables. You can further tweak the design in OpenSCAD before exporting to STL for 3D printing.

Contributing

This is an open-source project! You can contribute by:

  • Adding components: Sign in with GitHub and add new components with accurate measurements
  • Verifying dimensions: Help verify component measurements submitted by others
  • Reporting issues: Found a bug or have a feature request? Open an issue on GitHub
  • Contributing code: The application is open source - pull requests welcome!

Technology

Built with:

  • Python / Flask
  • HTMX + Alpine.js for interactive UI
  • GitHub API for component database storage
  • Nexar/Octopart API for component data import
  • OpenSCAD for 3D modeling
