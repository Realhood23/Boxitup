How It Works

1

Select Components

Search our database or add new components from DigiKey/Mouser with automatic dimension extraction.

2

Toggle Features

Enable the ports and connectors you need access to - USB, power, HDMI, GPIO, and more.

3

Place & Customize

Position components in the visual editor. Add ventilation, labels, and mounting options.

4

Generate & Print

Download your OpenSCAD script, render in OpenSCAD, and export STL for 3D printing.

Features

Component Database

Pre-measured components with accurate dimensions. Community-contributed via GitHub with version control.

Feature Toggles

Turn on/off USB, power, display, and other features per component. Only generate holes you actually need.

Full Customization

Multiple enclosure shapes, lid types, ventilation patterns, text labels, and mounting options.

OpenSCAD Output

Parametric, well-commented OpenSCAD scripts. Further customize in OpenSCAD's Customizer.

Auto-Import

Paste a DigiKey or Mouser URL to automatically extract component dimensions and specs.

GitHub Backed

Component database stored on GitHub. Contribute new components, track changes, and collaborate.
